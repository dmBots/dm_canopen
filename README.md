# CANopen 
## 项目简介
此项目是一个用于CANopen协议的字典工具，包含了CANopen通信协议及相关的测试和控制示例。

## 目录结构
CANopen字典工具/CANopenEditor-build: CANopen字典工具的构建目录

固件: 存放与项目相关的固件文件

字典: 存放CANopen协议的字典文件

控制例程: 存放控制相关的示例代码

CANopen协议移植及测试.pdf: CANopen协议的移植及测试文档

DM CANopen通信协议.md: DM CANopen通信协议文档

DM电机字典.md: DM电机相关的字典文档
