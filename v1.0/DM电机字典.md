## 1.对象字典

对象字典是CANopen网络中的核心，它是一个数据存储区域，用于管理节点的配置参数和状态信息。在CANopen网络中，每个节点都有自己的对象字典，其他节点可以通过访问对象字典来与该节点进行数据交换和配置。

### 1.1 对象分类说明

+ 索引：指定每个对象在对象字典中的位置，以十六进制来表示。
+ 子索引：指定变量对象在该索引下的位置，以十六进制表示。

#### 1.1.1 数据类型说明

| 数据类型 |        数据范围        | 数据长度 | DS301值 |
| :------: | :--------------------: | :------: | :-----: |
|   int8   |       -128~+127        |  1 byte  |  0002   |
|  int16   |     -32768~+32768      |  2 byte  |  0003   |
|  int32   | -2147483648~2147483647 |  4 byte  |  0004   |
|  uint8   |         0~255          |  1 byte  |  0005   |
|  uint16  |        0~65535         |  2 byte  |  0006   |
|  uint32  |      0~4294967295      |  4 byte  |  0007   |
|  string  |         ASCII          |    -     |  0009   |

#### 1.1.2 读写类型说明

| 读写类型 |   说明    |
| :------: | :-------: |
|    RW    |  可读写   |
|    WO    |   只写    |
|    RO    |   只读    |
|  CONST   | 常量 只读 |

#### 1.1.3 对象分类说明

| 类别 |         含义         | DS301值 |
| :--: | :------------------: | :-----: |
| VAR  |     单一简单数值     |    7    |
| ARR  | 具有相同类型的数据块 |    8    |
| REC  | 具有不同类型的数据块 |    9    |

|  索引  | 子索引 |     名称     | 可访问性 | 能否映射 | 数据类型 | 类别 | 数据范围 | 出厂设定 |
